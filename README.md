This repo contains all my dotfiles and a script I can deploy on a new machine.

- i3wm
- yay (for installing aur packages)
- i3blocks
- stow
- picom
- dunst
- feh
- alacritty
- keyd (aur)
- oldschool-pc-fonts (aur)
- awesome-terminal-fonts
- scrot
- adwaita-dark (aur)
- elementary-icon-theme
